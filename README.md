# SindhCoin Mining Website

## Deployment
1. Replace firebase-config.js with your Firebase keys.
2. Upload files to GitHub / Netlify / Firebase Hosting.
3. Test mining, referral, dashboard, withdraw functionality.
4. Optional: Convert to PWA or APK using AppGeyser.

## Features
- Daily Mining: 300 SINDH per day
- Referral Bonus: 1000 SINDH per referral
- Dashboard: Balance, Referrals, Total Mined
- Withdraw Request: Minimum 5000 SINDH
- Admin Panel: Approve/Reject withdraws
